# Avatar Images

This directory contains user avatar images for the admin dashboard.

For development, we're using initials-based fallbacks instead of loading actual image files.

To add real avatars:

1. Place avatar images in this directory (admin.jpg, user.jpg, etc.)
2. Update the avatar components to reference these files

## Current Status

- Using fallback initials for all avatars
- No 404 errors from missing image files
